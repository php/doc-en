xmldoc

(PHP4 >= 4.0b4)

xmldoc — Crea un objecto DOM representando un documento XML

Descripción

object xmldoc (string str)

Esta función analiza el documento XML contenido en el texto str y devuelve un objecto de clase "Documento Dom", que tiene las propiedades "doc" (resource), "version" (string) y "typo" (long).
